What is Appium?

Appium is an open source automation tool for running scripts and testing native applications, mobile-web applications and hybrid applications on Android or iOS.

Why use Appium?

Core Features

  • Open Source
  • Integration with Continuous Integration
  • No App Modification Required
  • Built-in Inspector for Element Identification
  • Real Device and Emulator/Simulator Testing
  • Supports Native, Hybrid and Mobile Web Applications
  • Wide Range of Appium Client Libraries

Appium Architecture

  • Appium Core
  • Drivers
  • Clients
  • Plugins
  • Appium Inspector
