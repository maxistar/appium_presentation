One Tool to rule them all...

Appium is an open-source project and ecosystem of related software, designed to facilitate UI automation of many app platforms, including mobile (iOS, Android, Tizen), browser (Chrome, Firefox, Safari), desktop (macOS, Windows), TV (Roku, tvOS, Android TV, Samsung), and more!

Why use Appium?

  • automation of web applications
  • automation mobile applications
  • automation desktop applications
  • automation of bots
  • automation of proprietary application
  • automation of lagacy applications

Core Features

  • Open Source
  • Integration with Continuous Integration
  • No App Modification Required
  • Built-in Inspector for Element Identification
  • Real Device and Emulator/Simulator Testing
  • Supports Native, Hybrid and Mobile Web Applications
  • Wide Range of Appium Client Libraries/Plugins/Drivers
